Some Useful Xpath Expressions

Given the following sample xml:

Find all elements

//*

Find all attributes

//@*

Find all elements with child

Find all child elements of bookstore, with grandchild elements

/bookstore/*[*/*]

Find last child element for each child of bookstore

/bookstore/*/*[last()]

Find last grandchild element for bookstore

(/bookstore/*/*)[last()]

Find all element from the my namespace

//my:book

Find publication price > 3

/bookstore[price > 3]

Find all attrs from my namespace

//@my:*

Find all elements with last-name or price as child

//*[price or last-name]

Find all elements without attributes

//*[not(@*)]

Find elements with text nobody

//*[. = "nobody"]

Get text of my:title

/bookstore/my:book/my:title/text()

Get val of attribute specialty of bookstore element

string(/bookstore/@specialty)

Get all first name or last name element

//first-name | //last-name

Get comment node

/bookstore/comment()

Select all node of magazine

/bookstore/magazine[1]/node()

Select everything of magazine

/bookstore/magazine[1]/node()|@*
